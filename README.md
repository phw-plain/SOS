# SOS(Survival Of Subway)

> C++언어로 만든 지하철 생존 시뮬레이션 콘솔 게임입니다.

* 기획·개발 기간 : 2021.08.17 ~ 2021.10.31 (약 2개월)

### GAME EXPLANATION

> "무너진 지하철에서 살아남기"라는 스토리를 가고 있습니다.

* 생존에 필요한 조건 <b>수분, 배고픔, 에너지</b>를 쾌적 상태로 유지하여야 합니다.
* 상점에서 아이템을 구매하여 상태를 회복할 수 있습니다.
